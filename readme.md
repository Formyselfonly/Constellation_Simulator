## 如何游玩?

1. cd到Constellation_simulator
2. pip install requirements.txt
3. 在.env里面输入自己的apikey
4. 运行你想模拟的星座
   1. 摩羯座   streamlit run Capricorn.py
   2. 狮子座   streamlit run Leo.py
   3. 处女座  streamlit run Virgo.py
   4. 天蝎座  streamlit run Scorpio.py
